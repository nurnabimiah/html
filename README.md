

## Branch: task-02-style

In this branch, I am using **HTML inline styles**.  
Here we have learned how to apply style using the `style` attribute.

### Example Code:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body style="background-color:red;">
  <h1 style="text-align:center; color:white;">Hello Mr Nurnabi</h1>
  <hr/>
</body>
</html>
